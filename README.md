# 🧩 The Maze Runner

## 📖 Description
-The Maze Runner is a game where the player controls a blue ball to navigate and escape a maze.  
-A hostile ball wanders through the maze and attempts to catch the player. Avoid the enemy while finding the fastest path to the goal!  
-Each playthrough generates a new random maze, making every game unique.

## 🎮 Gameplay
- Control a ball inside a maze
- Reach the goal to win the game
- Avoid the wandering hostile ball
- If the hostile ball catches you → Game Over

---
## 🕹️ Controls
| Key | Action |
|-----|--------|
| W / ↑ | Move Up |
| S / ↓ | Move Down |
| A / ← | Move Left |
| D / → | Move Right |
|  ESC  | Restart and change the maze|

---
## ⚙️ Features
- Each playthrough generates a new maze layout, making every game unique.
- The difficulty changes depending on the maze complexity, creating unpredictable challenges.
- A wandering bot moves through the maze and tries to catch the player.  
- Reach the goal before getting caught to win.
---
## 🚀 Installation
1. Clone the repository:
```bash
git clone https://github.com/11525036-ship-it/Game.git
cd maze-runner
pip install pygame
python main.py
```
3. How the game work
- A new maze will be generated each time play
- The maze layout changes the difficulty dynamically
- The game will continuously check:
  - If player reach the goal -> win
  - If the hostile ball catch player -> game over
